Today was: Data Recovery and Hardware Upgrades

Time to replace my HDD

The work done today

None on the site. Ended up staying up way too late last night backing up my HDD and getting the backup ready to be transferred to another HDD. My computer froze up and rebooted on it's own late last night. Got a warning about the HDD, so ran some tests and the consensus is time for a new HDD. Considering I've been using the same one since 2010, I can't say I'm surprised. I'm just happy there was a warning before actual failure, so I had time to create a new backup. I regularly save individual files on backup drives, but rarely make a full HDD backup. Normally I just replace an entire computer, but this rig has become Frankenstein's monster. I didn't realize I've had it since law school though and I'm pretty impressed with how well it has kept up.

So today is studying and Udemy videos. Their web dev bootcamp series has been a pretty solid introduction to the subject. I just finished the front end videos and now it's on to the backend. I'm sure I'll learn dozens of things which I will want to add to the site, or I'll learn how to do what's already up in a better way.
