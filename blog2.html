Disaster Averted
Aug 13, 2017 / by Robert C. Farrell

New HDD installed and working. It was much more work getting it up and running than I anticipated. The hard drive with bad sectors was preventing a full clone of the data on it, so I had to set up the new HDD as the main HDD manually. The first attempt was successful, but the hard drive was only showing 500GB out of 2TB available. The only solution I found was reformatting and reinstalling Windows. Which was done and now the PC boots to the new harddrive, shows the correct size and runs so much better. I'm pretty surprised as how much the old HDD was slowing down my system, mostly in the form of lag when opening programs. Almost feels like a new computer.
