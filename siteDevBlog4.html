Off Work Wednesday

August 9, 2017 / by Robert C. Farrell

Today was: Bug fixes

The IRA contribution calculator was the focus today. Plus a page addition.

The work done today

Wednesday is supposed to be a slow day. Since I don't really have weekends, I like to take Wednesday's pretty easy so not much work was done today. Just an minor bug fix on the Resume page. The first page of this site was my resume so it's mostly html with the just the 'Open All' button having any javascript behind it. This bit of code was the first javascrupt I wrote, and was a simple class toggle for the duties description sections.

The big fix was for this toggle. The list of jobs and descriptions were made with Bootsrap, and the work descriptions are in Boostrap dropdown menus. The toggle all button was originally just adding a class which hid the descriptions. The bug happened when the toogle all would be showing the description while the Boostrap toggle was trying to hide them. This cause the descriptions to show while the Boostrap menu was collapsed, and vice versa.

The knowledge I gained working on other parts of the site made fixing this a breeze and appears to be working as intended. The fix was just putting a check in to look for the Boostrap classes and change them if needed. Some minor style changes were done too just to distinguish the Dev Blog from the Tax Blog. There is still so much to do, but today is more job hunting, studying and some Heroes of Might and Magic 6.

The Big Problem of the Day

Today's big problem was with project scope, and life. Working on this site has been a great experience, but the possibilities seem endless. It is easy for me to get so into improving this site that I lose focus on other things. So today will be job junting, studying for Reg, and some HoMM6. Instead of getting the calculator working, the resume page more interactive, getting all my articles proofed and posted on the blog, breaking ground on the other calculators, getting cohesion and depth to the style, ad naseum.
